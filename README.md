# SENTINEL AI Spline Hero Demo

A compact public demo showing a full-screen dark security-company landing page with an embedded Spline 3D background.

## Live References

- Green: https://afinagen.github.io/sentinel-ai-spline-hero-demo/
- Gold: https://afinagen.github.io/sentinel-ai-spline-hero-demo/?variant=gold

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui-style Button component
- `@splinetool/react-spline`
- `@splinetool/runtime`

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm test
npm run lint
npm run build
```

The live demo is intended to be deployed through GitHub Pages.
